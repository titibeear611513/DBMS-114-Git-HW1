# DBMS 114 Git Homework

This repository contains the Week1 TA HW1 assignment for learning Git workflow.

## Tasks

Complete the following 6 tasks:

### Task 1: Fork Repository

Fork this repository:
```
https://github.com/113356041-bot/DBMS-114-Git-HW-Repo
```

### Task 2: Clone Repository

Clone your forked repository to your local machine.

### Task 3: Branching & Edit

1. Create a branch named `hw/<student-id>`
   - Example: `hw/112356040`

2. Create a file named after your student ID
   - Example: `112356040.txt`

3. In the file, write down your student ID
   - Example: `112356040`

### Task 4: Make a Commit

1. Commit your changes
2. Push to origin with branch named `hw/<student-id>`
   - Example: `hw/112356040`

### Task 5: Create a Pull Request

Create a Pull Request (PR) with the following settings:
- Base branch: `main`
- Compare branch: `hw/<student-id>`

### Task 6: Upload Screenshot

1. Take a screenshot of your Pull Request
2. Submit a PDF file containing:
   - Your screenshot
   - Your group members' screenshots

## Example

If your student ID is `112356040`:

- Branch name: `hw/112356040`
- File name: `112356040.txt`
- File content: `112356040`
